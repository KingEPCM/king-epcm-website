Generate employment letter HR

Enter the employee's details and download a signed, branded King EPCM employment / income-verification letter. The letter is dated today.

HR only

This tool is for the HR team. If you should have access, email it@kingepcm.com to be added to the HR role. To request a letter for yourself, use the request page.

Employee details

Adds the wording ServiceOntario requires: confirms full-time employment, the position, the start date, and that King EPCM intends to employ the person for at least six months. Position and start date become required. Add the employee's Ontario address below if the letter is also for proof of residency.

Signer defaults to Angela Shi — change it if someone else is signing. Position, start date, date of birth, and pay are optional; only what you fill in appears in the letter. Print the PDF and sign above the name.
